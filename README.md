# Pearlville Redesigned GitHub Pages Site

This version turns the earlier Pearlville concept into working HTML, CSS, and JavaScript.

## Included

- Full-width hero photograph
- Large “Real People. Real Things.” headline
- Decorative Pearlville wordmark
- Pink and cream content panels
- Inline funding and program icons
- Five-part vision section
- Funding and partnership section
- Email signup placeholder
- Strong footer
- Desktop, tablet, and mobile layouts

## Upload to GitHub

Replace the old files in your `PEARLVILLE` repository with:

- `index.html`
- `styles.css`
- `script.js`
- the entire `assets` folder

The image must stay at:

`assets/pearlville-hero.jpg`

Otherwise, the hero photograph will not appear.

## Replace before public launch

Search `index.html` for these placeholders:

- `https://forms.google.com`
- `hello@pearlville.org`
- `href="#"` social links
- The email signup form, which is visual only

## Important language

The website identifies Pearlville as a developing initiative and labels the programs as proposed. Keep that language until programs, funding, partnerships, and legal status are established.
