# Pearlville GitHub Pages Mockup

This starter website presents Pearlville as a **new community initiative in development**.

## What is included

- Responsive homepage
- Founding story
- Proposed programs clearly labeled as future plans
- Funding, sponsorship, and partnership section
- Founding volunteer interest call-to-action
- Mobile navigation

## Before publishing

1. Replace the Google Forms placeholder link in `index.html`.
2. Replace the sample email address and social links.
3. Edit the wording to match your final mission.
4. Add your own logo or photos later.
5. Do not claim nonprofit or tax-exempt status unless Pearlville has formally received it.

## Publish with GitHub Pages

1. Create a new public GitHub repository, such as `pearlville`.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then click **Save**.
6. GitHub will provide your public website link.

The mockup does not include fake impact statistics, completed events, or claims of existing funding.
